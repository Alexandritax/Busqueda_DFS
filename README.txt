El programa main.rb es el programa principal

Para correr el codigo debe de tener el compilador de ruby: https://rubyinstaller.org/ y se uso como editor Visual Studio Code

El codigo funcionara si y solo si lee la matriz de adyacencia del grafo con las siguientes condiciones
    -Los elementos solo son 0's o 1's
    -Cada 0 o 1 debe de estar separado por tabulador o \t
    -Cada linea debe estar separado por un salto de line o \n
    -La matriz debe ser una matriz cuadrada simetrica con los elementos de la digonal principal siendo 0.
